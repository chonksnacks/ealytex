# Ealytex Global Inc — Website

Official website for Ealytex Global Inc., a U.S.-based fashion manufacturer headquartered in New York City.

## Structure

```
ealytex-site/
├── index.html       # Main site (single page)
├── images/          # All logo and asset images
│   ├── *.png        # Client/partner logos
└── README.md
```

## Hosting on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://yourusername.github.io/repo-name/`

## Notes

- All styles and scripts are self-contained in `index.html`
- External images (hero, about, factory) still reference Wix CDN — replace with your own hosted images when ready
- Fonts loaded from Google Fonts (requires internet connection)
